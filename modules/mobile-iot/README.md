# Mobile & IoT Support

The **Mobile & IoT module** empowers OpenT3Base with real-time, on-the-go, and connected operations via mobile devices and industrial IoT.

## Key Features

- Native and cross-platform mobile apps (iOS, Android, tablets)
- Mobile-optimized interfaces for all modules (shop floor, warehouse, sales, management)
- IoT device SDKs and gateway APIs (MQTT, OPC-UA, REST)
- Real-time data collection from sensors, machines, and trackers
- Edge computing and offline-first capabilities
- Push notifications and alerts

## Integration

- Mobile apps for MES, Logistics, SCM, CRM, and more
- Real-time IoT data feeds into MES, SCM, BI, and QMS
- APIs and SDKs for mobile and device developers

## Role in OpenT3Base

Enables users and devices to interact with the system wherever they are, driving digital transformation at the edge and on the move.
