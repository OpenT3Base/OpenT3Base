# PM (Project Management)

The **PM module** enables teams to plan, track, and deliver projects—engineering, R&D, or operations.

## Key Features

- Project and milestone planning
- Task assignments and scheduling
- Resource allocation
- Gantt charts and Kanban boards

## Integration

- **Links with** PLM, ERP, and DMS for project-related data.

## Role in OpenT3Base

PM drives execution, collaboration, and delivery across teams.
