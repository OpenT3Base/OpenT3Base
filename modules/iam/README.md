# IAM (Identity & Access Management)

The **IAM module** secures OpenT3Base, managing users, roles, and permissions.

## Key Features

- Single Sign-On (SSO)
- Role-based access control
- Multi-factor authentication
- Audit logging

## Integration

- **Secures** all modules with unified identity and access policies.

## Role in OpenT3Base

IAM keeps your platform safe, compliant, and user-friendly.
