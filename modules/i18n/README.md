# Localization & Internationalization (L10n & I18n)

The **Localization & Internationalization module** ensures OpenT3Base can serve users worldwide, adapting every module to local languages, cultures, currencies, and regulatory needs.

## Key Features

- Multi-language support for all user interfaces, reports, and documents
- Locale-aware date, time, and number formatting
- Currency, tax, and unit of measure conversions
- Pluggable translation files and runtime language switching
- Regional compliance, e.g., GDPR, tax
- Right-to-left (RTL) and left-to-right (LTR) script support

## Integration

- All modules consume localization services for a seamless, consistent user experience
- Central translation management for contributors
- APIs and SDKs for third-party/localization integration

## Role in OpenT3Base

Makes OpenT3Base truly global and accessible, supporting users in any language or region.
