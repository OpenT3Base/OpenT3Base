# MES (Manufacturing Execution System)

The **MES module** orchestrates all activities on the shop floor, from raw material to finished product.

## Key Features

- Production order management
- Work center scheduling
- Real-time production tracking
- Quality checks and traceability
- Integration with IoT devices

## Integration

- **Receives** production plans from MRP.
- **Reports** execution, yield, and quality to ERP, QMS, and BI.

## Role in OpenT3Base

MES drives operational excellence and real-time visibility in manufacturing.
